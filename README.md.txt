# County Research Agent v1

Alpha v3 public viewer for county documents.

## What this does

- Search parsed county documents
- Open and preview documents
- Download parsed text
- Download original PDFs
- View PDFs inline

## How it works

This is the public viewer layer only.

Documents are:
- harvested locally
- parsed locally
- stored in data/raw and data/parsed

This app lets anyone browse and search them.

\# County Research Agent v1



\## Purpose



This tool is designed to make public county records easier to access, search, and understand.



All documents displayed in this application are public records sourced from official county publications, including agendas, minutes, and supporting documents.



The goal is simple:



\- Help the public find information faster  

\- Make large documents easier to navigate  

\- Improve transparency and accessibility  

\- Reduce friction in understanding local government activity  



\## What this app does



\- Search across parsed county documents  

\- Open and preview records quickly  

\- Download original PDFs  

\- View clean text versions of documents  



\## Important note



This application does not create or alter public records.



It only reorganizes publicly available information to make it easier to access and explore.
